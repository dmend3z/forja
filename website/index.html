Quick Start

Or: brew install dmend3z/forja/forja

2

Initialize — all 25 agents installed

forja init

Start building

forja plan "add user auth with JWT"

With a team: forja task "..." --team full-product

Why forja?

Everything you need, nothing you don't

Claude Code supports agents and skills — but managing them across projects is manual. forja gives you a single CLI to install, search, and manage all of it.

25 Curated Agents

Organized into 5 dev phases. Research, code, test, review, and deploy with purpose-built agents for each step.

Agent Team Configs

Multi-agent development pipelines. From full 5-agent product teams to quick 2-agent hotfix crews.

One CLI

Install, uninstall, search, and manage everything with a single command. No manual file copying or configuration.

Symlink-Based

Skills live in a central registry, installed via symlinks into ~/.claude/agents/. Clean, reversible, zero duplication.

CLI Commands

Simple, powerful commands

Manage your entire skills setup from the terminal. Install, search, plan, and deploy with one tool.

Terminal 
# Setup
forja init                          # Initialize registry
forja doctor                        # Health check

# Browse
forja phases                        # Show the 5 workflow phases
forja list --available              # Show all available skills
forja search typescript             # Search by keyword
forja info code/rust/feature        # Skill details

# Install & Manage
forja install test/tdd/workflow    # Install a skill
forja install --all                # Install everything
forja uninstall test/tdd/workflow  # Remove a skill
forja update                        # Update the registry

# Plan & Execute
forja plan "add auth to the API"    # Create implementation plan
forja execute                       # Execute the latest plan
forja task "fix login bug"          # Run task directly

# Teams
forja team preset full-product    # Create team from preset
forja team list                    # List configured teams

How It Works

Simple symlink architecture

No magic. forja clones a skills registry and creates symlinks. Claude Code picks up the agents automatically.

1

Clone Registry

forja init clones the skills repo into ~/.forja/registry/

2

Auto-Install Agents

forja init auto-installs all agents as symlinks into ~/.claude/agents/

3

Auto-Detected

Claude Code discovers the new agents automatically. No restart needed.

4

Clean Uninstall

forja uninstall removes symlinks and updates state. No leftover files.
